# 防骗反诈手册（Scam Shield）

> 投资圈里**少亏 = 多赚**。识别骗局的能力比选股选基重要 10 倍。

## 一、识别骗局的「3 句口诀」

任何投资请求出现下列任一关键词，立刻拉响警报：

1. **"保本保息" / "稳赚不赔" / "零风险高收益"** —— 这话只要出现，无论嘴是谁，99% 是骗局。**合规金融产品禁止此类宣传**。
2. **"年化 > 8% 还说稳健"** —— 银行理财稳健的天花板就是 4%-5%，超过 6% 你要担心本金，超过 10% 准备损失全部。
3. **"内部消息 / 拉你进群带单 / 老师手把手教"** —— 真有内部消息的人不会带不认识的你赚钱。

## 二、13 类常见骗局（按出现频率从高到低）

### 1. 杀猪盘（情感投资类）

- **套路**：网恋/微信交友 → 培养感情 → 透露"我有内部渠道，每月能赚 30%" → 拉你下载假平台 → 前期小赚让你充值 → 大额投入后无法提现 → 拉黑跑路。
- **关键识别**：
  - 没见过面但聊得火热
  - 推送陌生 App（非银行/券商/淘宝/微信支付等大厂）
  - 声称小赚后劝你"再加大投入"
- **防范**：网恋不谈投资，谈投资必见面；陌生 App 一律不下载。

### 2. 庞氏骗局（用新钱付旧息）

- **套路**：承诺远高于市场水平的固定收益（年化 12%-30%），用新投资人的钱付老投资人的"利息"，制造"真能赚钱"的假象。
- **历史案例**：e租宝、钱宝网、原油宝、某某养老社区、某某矿池。
- **关键识别**：
  - **收益结构不可解释**（说不清钱具体投了什么）
  - 持续承诺超额固定收益
  - 推荐拉人头返佣
  - 所谓"国资背景""政府支持"无法在工商注册官网验证

### 3. 原始股 / Pre-IPO 骗局

- **套路**：声称某公司即将上市，现在内部购买"原始股"，上市后 10 倍 100 倍。
- **真相**：
  - **真原始股有 36 个月以上的限售期**，不会让普通投资者轻易买
  - 99% 的"原始股"实际是骗子虚构的，公司根本没有上市计划
  - 即便公司是真的，"上市"也可能拖延或失败，钱被锁死
- **防范**：在中国证监会官网"行政许可信息平台"查证发行计划；非合格投资者（个人金融资产 < 300 万）依法不能买非公开发行股权。

### 4. 高息 P2P / 民间借贷

- **现状**：国内合规 P2P 已全面清退（2020 年）。任何宣称"P2P 投资"的现在都是骗局。
- **变种**："消费金融 App""数字资产理财""供应链金融"

### 5. 虚拟币山寨盘

- **套路**：发个连白皮书都没有的"币"，拉群喊单、操控价格、收割韭菜后跑路。
- **国内法律风险**：央行多次发文禁止虚拟货币交易，**没有任何法律保护**。
- **真比特币的合规途径**：仅在合法监管的交易所（如美国注册的）可购买，且仓位 < 5%。

### 6. 假冒券商 / 基金公司 App

- **套路**：仿冒招商证券、易方达基金等 App，名字字体几乎一样。引诱用户充值。
- **识别**：
  - 必须从**官方网站**或**应用市场官方账号**下载
  - 真券商资金账户与第三方银行账户存管，钱不会进 App 公司账户
  - 国家反诈中心 App 自动检测可疑应用

### 7. 假基金经理 / 假私募

- **套路**：朋友圈晒"操盘手"身份，给"幸运儿"开放限量名额，号称"年化 50% 的私募份额"。
- **识别**：所有合法私募基金管理人在中国证券投资基金业协会（AMAC）官网都能查到登记号。**先去 https://gs.amac.org.cn 查机构登记**。

### 8. "炒股大师"付费课程 / 荐股 VIP 群

- **套路**：免费群讲两次"准"了，引你交 9888 / 39800 进 VIP 群"独家牛股"。结果 VIP 群里推荐的票要么早就涨到山顶，要么是"庄家出货群"。
- **真相**：
  - **国内荐股需要持有"投资咨询业务"牌照**，可在证监会官网查询。
  - 真正赚钱的私募/公募经理基本不会公开荐股，因为容易引发监管风险。

### 9. "代客理财" / 全权委托账户

- **套路**：把账户交给别人操作，并保证收益。
- **真相**：法律明确禁止"代客理财"。一旦发生亏损，受害人很难追回，**且本人需对内幕交易等违法行为承担连带责任**。

### 10. 区块链 / NFT / 数字藏品骗局

- **套路**：包装成"数字资产""文化艺术品"，配合"拉新返佣"。
- **现状**：国内大量"数字藏品平台"已暴雷或被监管处罚。**99% 不具备投资价值**。

### 11. 外汇黄金期货平台

- **套路**：在境外注册一个看似有"美国 NFA / 英国 FCA"监管的平台，实则数据自己控盘，让你 100% 亏损。
- **真相**：**中国大陆境内不允许私人参与外汇保证金、贵金属保证金、外汇期货等交易**，凡是说能做的都不合法。

### 12. 养老 / 健康产品理财骗局

- **目标人群**：老人
- **套路**：免费体检、礼品、旅游 → 推销"养老社区""健康床位""保健品理财" → 老人投入毕生积蓄 → 公司跑路。
- **特征识别**：让老人把钱直接转入私人账户、要求购买"会员卡"换取分红。

### 13. AI 量化 / 机器人智能交易

- **2024-2025 年新型骗局**：宣称用 AI/量化机器人保证赚钱，月收益 20%。
- **真相**：真正的量化基金年化 15%-25% 已是顶尖水平，且回撤动辄 -10%；个人小账户不可能享受机构级量化收益。

## 三、5 步「持牌识别法」（核实合法性必备）

任何投资产品先做这 5 件事：

### Step 1：查机构是否持牌
- **银行**：中国人民银行 / 银保监会
- **券商**：中国证监会官网
- **公募基金**：基金业协会（AMAC）
- **私募基金**：AMAC 私募登记
- **保险**：银保监会保险机构信息

### Step 2：查产品是否在册
- 银行理财：在中国理财网（chinawealth.com.cn）查产品登记编码
- 公募基金：在基金公司官网或 AMAC 查代码
- 私募基金：在 AMAC 查产品备案编号

### Step 3：查合同是否合规
- 看到"保本保息""固定收益""零风险"字样 → 立即终止
- 注意"业绩比较基准"不是"承诺收益"

### Step 4：查资金是否托管
- 合规：钱进入第三方银行存管账户，不进入公司账户
- 不合规：钱直接打到对方公司账户，甚至个人账户

### Step 5：查投诉记录
- 黑猫投诉、聚投诉
- 中国裁判文书网（搜公司名 + "非法集资"等关键词）
- 各地金融监管局官网披露的非法金融机构名单

## 四、被骗后怎么办

1. **立即停止汇款**，截图所有沟通记录、转账记录、合同。
2. **拨打 110 报警**，并到当地公安经侦部门报案。
3. **下载"国家反诈中心" App**，提交线索。
4. **不要轻信"维权群"**——里面 80% 是二次诈骗（声称能帮你追回钱，再骗一次）。
5. **维权周期长，做好心理准备**。庞氏骗局类追回率通常 < 10%。

## 五、给家里长辈的「3 不原则」

- **不贪图高息**：超过 5% 的产品坚决不碰
- **不签陌生合同**：所有"理财"都让子女先看
- **不下载非银行 App**：手机里只装 6 大行 + 持牌券商 App

## 六、推荐的合规渠道

- 银行 App（六大行 + 招行/兴业等）：适合保守型
- 蚂蚁财富 / 天天基金 / 京东金融：适合买公募基金，**只支持公募，不会有奇怪产品**
- 同花顺 / 雪球 / 东方财富 / 富途（港美股）：适合炒股，但选**官方代销/券商对接版**
- 国债：建行/工行/中行 App "国债"专区，或直接在中国债券信息网

## 七、最后忠告

**金融的本质是用风险换收益。任何"无风险高收益"的承诺都是反金融常识的。**

如果你身边有人正在被骗，不要争论"这是不是骗局"，而是直接给他看：

- 这个机构在 AMAC / 证监会官网**有没有备案？**
- 这个产品的**底层资产是什么？讲不清就跑。**
- 它声称的**年化收益率是不是高得离谱？**

记住：**让你心动的高收益，往往瞄准的是你的本金。**
