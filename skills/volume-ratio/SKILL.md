# 量比指标分析（Volume Ratio）

## 概述

量比是 A 股短线交易中最直接反映主力资金动向的核心指标，连徐翔都讳莫如深。它衡量当日每分钟成交量与过去 5 日同一分钟平均成交量的比值，能瞬间看穿主力吸筹、洗盘、拉升、出货的真实意图。

## 核心公式

```
量比 = 当日每分钟成交量 / 过去5日同一分钟平均成交量
```

- **基准线**: 量比 = 1，表示当日成交量与近 5 日持平
- **关键阈值**: 量比的大小直接决定主力是「在干活」还是「在看戏」

## 五档量比区间与实战指令

加载 `references/rules.md` 获取完整规则表。以下是速查版：

| 量比区间 | 信号强度 | 核心指令 |
|----------|---------|---------|
| **< 0.6** | 极度缩量 | 无主力关注，短期无大行情，**直接放弃** |
| **0.9 - 1.5** | 常规波动 | 多空不明，无参考价值，**不进场** |
| **1.5 - 2.5** | 温和放量 | 上升趋势=持仓；下跌趋势=诱多陷阱，**离场** |
| **2.5 - 5** | 明显放量 | 关注压力位，大概率突破，**拿住筹码** |
| **> 5** | 巨量放量 | 低位=主力吸筹，**上车**；高位=主力出货，**立刻离场** |

## 使用方式

1. 当分析某只股票时，获取其实时或历史量比数据
2. 对照五档区间规则，判断量比所处的信号区间
3. **结合当前价格位置**（高位/低位/压力位/支撑位）给出明确操作指令
4. 输出格式：量比数值 -> 所属区间 -> 信号解读 -> 操作指令

## 与价格位置的联动判断

量比必须结合价格位置才有意义：

- **低位 + 量比 > 5**: 主力底部吸筹，黄金买点
- **高位 + 量比 > 5**: 主力高位出货，逃命信号
- **压力位 + 量比 2.5~5**: 放量突破压力位，压力变支撑
- **上升趋势 + 量比 1.5~2.5**: 健康放量上涨，持仓不动
- **下跌趋势 + 量比 1.5~2.5**: 主力诱多出货，别上当

## 注意事项

- 量比是辅助指标，需结合价格趋势、均线、K 线形态综合判断
- 开盘前 30 分钟量比波动剧烈，参考价值打折
- 尾盘突放大量通常有猫腻（主力对倒或次日砸盘信号）
