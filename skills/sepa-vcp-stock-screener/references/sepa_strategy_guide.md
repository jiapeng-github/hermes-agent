# SEPA 策略与 VCP 形态参考手册

> 基于马克·米勒维尼《股票魔法师》（Trade Like a Stock Market Wizard）核心理论整理

---

## 一、SEPA 策略概述

**SEPA（Specific Entry Point Analysis）** 是米勒维尼用来识别处于最佳买入窗口的成长股的综合分析框架，核心逻辑是：

> **基本面优秀 + 价格趋势向上 + 催化剂存在 + 低风险入场点**

四个维度同时满足，才进行交易。

### 四维筛选体系

| 维度 | 英文 | 含义 |
|------|------|------|
| 趋势（Trend） | T | 股价处于长期上升趋势中 |
| 基本面（Fundamentals） | F | 营收/利润高速增长，ROE 强劲 |
| 催化剂（Catalyst） | C | 新产品、新行业、政策利好等 |
| 入场点（Entry） | E | VCP形态中的最低风险入场位置 |

## 二、VCP 形态详解

**VCP（Volatility Contraction Pattern）波动收缩形态** 是 SEPA 策略中识别最低风险买入点的核心工具。

### 形态结构

```
价格
  |
  |    /\        
  |   /  \  /\   
  |  /    \/  \ /\___  ← 波动收缩
  | /          \/    \___  ← 突破点（买入）
  |________________________ 时间
  
  阶段1     阶段2   阶段3   突破
  大幅波动  收窄    再收窄  放量突破
```

### VCP 关键特征

1. **至少2-3次回撤收缩**  
   每次回撤幅度比上一次小（如：20% → 12% → 6%）

2. **成交量配合**  
   - 回调时：成交量萎缩（机构持股，无人抛售）
   - 突破时：成交量放大（机构建仓，资金涌入）

3. **时间周期**  
   典型 VCP 形态持续 3-8 周，过短为假信号，过长可能失效

4. **均线结构**  
   形态结束前股价需回到 MA50 以上，MA50 > MA150 > MA200

5. **基准区域（Base）**  
   整个收缩形态称为"基准"，突破基准最高点即为买入信号

### 买入规则

- **买入点**：价格突破基准最高点（枢轴点）时买入
- **仓位**：初始仓位不超过总资金的 25%，突破后加仓
- **止损**：严格设置在枢轴点以下 2-3%（米勒维尼原则：亏损绝不超过 7-8%）

## 三、A股 SEPA 筛选条件对照表

| 条件 | 米勒维尼原则 | A股对应阈值 |
|------|-------------|------------|
| 营收增长 | 季度营收同比加速 | > 25% |
| 净利润增长 | 季度净利润同比 + 加速 | 同比 > 30%，环比正增长 |
| 股价趋势 | 高于 50/150/200 日均线 | 高于 MA50 & MA150 |
| 成交量 | 突破时放量 / 回调时缩量 | 近10日均量 > 120日均量 |
| ROE | 机构级盈利质量 | > 15% |
| 净利润 CAGR | 三年持续增长 | > 20% |
| 股票资质 | 流动性良好的成长股 | 非ST，上市满1年 |

## 四、注意事项

### A股特殊性
1. **涨跌停制度**：A股有 10% 涨跌停限制，VCP 突破日可能无法以理想价格买入
2. **退市风险**：ST股随时可能退市，务必优先剔除
3. **季报质量**：A股季报信息披露质量参差不齐，建议以年报和半年报数据为主进行验证
4. **北向资金**：机构持仓中需关注北向资金动向，作为额外参考信号

### 适合本策略的股票特征
- 市值：50-500亿中小盘（流动性足够，机构未过度持仓）
- 行业：新能源、医疗、消费、科技等高成长赛道
- 概念：有明确的业绩驱动因素（非纯主题炒作）

## 五、常见假突破识别

| 信号 | 描述 | 处理建议 |
|------|------|----------|
| 成交量不足 | 突破时成交量 < 均量的 1.5 倍 | 观望，不急于介入 |
| 突破后快速回落 | 突破当日或次日收阴 | 果断止损 |
| 基本面未验证 | 只有技术形态，无盈利支撑 | 降低仓位或跳过 |
| 指数大幅下跌 | 整体市场趋势向下 | 暂停操作，等待市场企稳 |
